О проекте
=========

Этот проект разработан для автоматизации составления расписания в университетах. Основной модуль `data_preprocessing` отвечает за предобработку данных, необходимых для работы нейросети.

Начало работы
-------------

1. Установите зависимости, указанные в :doc:`/user_guide`.
2. Подготовьте данные в формате CSV.
3. Используйте модуль `data_preprocessing` для обработки данных.
